# 4λ Web

The landing page for [4Lambda](https://4lambda.io) — a static site built with Vite and deployed to Cloud Run.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview    # Preview the production build locally
```

## Deployment

Pushes to `main` automatically build and deploy to Cloud Run via GitHub Actions.

**Project:** `sylvan-triumph-161508`
**Service:** `web`
**Region:** `us-central1`
